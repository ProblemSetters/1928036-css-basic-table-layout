<p>It contains the following components.</p>

<ul>
	<li>A table with exactly 3 columns and 5 rows (excluding the header).</li>
</ul>

<p>Position the components as described.</p>

<p>Header Row:</p>

<ul>
	<li>The header should have a dark background color (<code>#333</code>).</li>
	<li>The text color should be green (<code>#1ba94c</code>).</li>
	<li>The text size should be 16px.</li>
	<li>The header text should be bold.</li>
</ul>

<p>Table Rows:</p>

<ul>
	<li>Odd-numbered rows should have a background color of <code>#ffffff</code> (white).</li>
	<li>Even-numbered rows should have a background color of <code>#f7f7f7</code> (light gray).</li>
	<li>The font size of the text should be 14px.</li>
	<li>The text color should be black (<code>#000</code>).</li>
</ul>

<p>Table Layout:</p>

<ul>
	<li>All cells should have a border of <code>1px solid light gray</code> (<code>#ddd</code>).</li>
	<li>All cell text should be left-aligned.</li>
</ul>

<p>The markup for the question is provided.&nbsp;Complete the <em>src/css/style.css</em>&nbsp;file to implement the required features.</p>
