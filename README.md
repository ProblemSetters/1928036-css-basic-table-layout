<p>The layout contains the following components:</p>

<ul>
	<li>A table with exactly 3 columns and 5 rows (excluding the header).</li>
</ul>

<p>Style all components as follows and position all components as per the UI provided:</p>

<p>Table Layout:</p>

<ul>
	<li>
	<article data-scroll-anchor="true" data-testid="conversation-turn-97" dir="auto">
	<p>There should be a 50px margin surrounding the table.</p>
	</article>
	</li>
</ul>

<p>Header Row:</p>

<ul>
	<li>The header should have a dark background color (#333).</li>
	<li>The text color should be green (#1ba94c).</li>
	<li>The text size should be 16px.</li>
	<li>The header text should be bold.</li>
</ul>

<p>Table Rows:</p>

<ul>
	<li>Odd-numbered rows should have a background color of #ffffff&nbsp;(white).</li>
	<li>Even-numbered rows should have a background color of #f7f7f7&nbsp;(light gray).</li>
	<li>The font size of the text should be 14px.</li>
	<li>The text color should be black (#000).</li>
</ul>

<p>Table Cells:</p>

<ul>
	<li>Each cell should have a thin border with a thickness of 1px, styled as a solid line, and use a light gray color (#ddd).</li>
	<li>All cell text should be left-aligned.</li>
</ul>

<p>The goal is to match the UI with the design shown in the demo.</p>

<p>The markup for the question is provided.&nbsp;Complete the <em>src/css/style.css</em>&nbsp;file to implement the required features.</p>
